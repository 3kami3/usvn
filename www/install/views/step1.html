What is USVN ?

USVN is an Open Source free software allowing everybody to use Subversion in an easily way.

How to make an easy use of SVN?

USVN is a web application with an easy interface to do every SVN configuration like user, access, management or hook configuration. Hooks are specific to each repository and let us do some interesting actions like deny a commit with a blank message or checking syntax of PHP file. USVN has two different running modes:

  • Integration into the official SVN server hooks.
  • His own SVN protocol implementation.

For each mode, USVN provides the same functionnalities :

  • SVN repositories management
  • A plugin manager for extend USVN features like a wiki, bug tracker, anything.

USVN is compatible with the original SVN protocol and is available on LAMP plateform.
